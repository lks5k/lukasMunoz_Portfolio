Mi proceso

Reflexión de mi camino
como desarrollador Java

Hace poco más de un mes tomé una de las decisiones más retadoras de mi vida: dejar atrás 10 años como Jefe de Diseño y Operaciones para empezar desde cero en el mundo del desarrollo de software.

No fue fácil. Sentarse a escribir las primeras líneas de código cuando estás acostumbrado a liderar equipos y resolver problemas con las manos es un cambio de mentalidad enorme. Pero rápidamente me di cuenta de algo: la lógica que usaba para optimizar procesos industriales es la misma que necesito para escribir código limpio y funcional. Solo cambia el lenguaje.

En estas semanas con Generation Colombia he aprendido HTML5, CSS, JavaScript, SQL y estoy construyendo mis bases en Java. Cada pequeño proyecto que funciona — por sencillo que sea — me confirma que tomé la decisión correcta.

Áreas que sigo trabajando
  • Profundizar en Java y programación orientada a objetos
  • Ganar fluidez en la lógica de programación
  • Construir proyectos reales que pueda mostrar con orgullo
Mis metas
  • Corto plazo: Terminar el Bootcamp con proyectos sólidos y conseguir mi primera oportunidad como Desarrollador Full Stack Java Jr.
  • Largo plazo: Crecer hacia roles de mayor responsabilidad técnica, aportando también mi experiencia en procesos industriales al diseño de soluciones de software.

Este portafolio es mi primer paso. Lo iré actualizando con cada proyecto nuevo, con cada error que aprenda a resolver y con cada versión mejorada de mí como desarrollador.
